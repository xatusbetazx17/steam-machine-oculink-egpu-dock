# Build Checklist (Printable)

## Phase 0 — Compatibility
- [ ] Steam Machine has M.2 Key‑M NVMe slot (PCIe lanes)
- [ ] Steam Machine can boot from USB storage (if needed)
- [ ] You have a storage plan (external USB‑C NVMe or second internal drive)
- [ ] You accept warranty/damage risks

## Phase 1 — Parts verification
- [ ] OCuLink connector types match across adapter/cable/dock
- [ ] PSU has correct GPU power connectors (8‑pin / 12VHPWR)
- [ ] Enclosure supports your GPU size class
- [ ] Fans + dust filters are planned

## Phase 2 — Steam Machine internal mod
- [ ] Power down / unplug / ESD safety
- [ ] Install M.2 → OCuLink adapter
- [ ] Connect short internal pigtail
- [ ] Install panel-mount OCuLink port
- [ ] Add strain relief
- [ ] Verify no airflow blockage
- [ ] Close case without crushing cable

## Phase 3 — Dock build
- [ ] Mount PSU securely
- [ ] Mount dock board securely
- [ ] Install GPU and secure bracket
- [ ] Add anti-sag support if needed
- [ ] Install intake + exhaust fans
- [ ] Install dust filters
- [ ] (Optional) Mount USB‑C NVMe enclosure away from GPU exhaust
- [ ] Cable management: tie down cables, avoid fan contact

## Phase 4 — Software & first boot
- [ ] (If needed) Install SteamOS on external NVMe
- [ ] Monitor connected to dock GPU (HDMI/DP)
- [ ] Power sequence: dock PSU ON → Steam Machine ON
- [ ] Verify GPU detection

## Phase 5 — Validation
- [ ] 10 repeat boots successful
- [ ] 20–40 min gaming test stable
- [ ] Temps within target range (no sustained throttling)
- [ ] Filters accessible for cleaning
